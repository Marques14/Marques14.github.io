Desafio de Programação Sobre Listas e Dicionários

Habbo Hotel - Fiscal dos Fiscais

  • Joab Henrique
  • Programação
  • 26 Out 2024

Imagine que você é o responsável por manter a ordem no coração de uma corporação policial dentro do icônico mundo virtual do Habbo Hotel, em um ambiente onde cada decisão pode ser a linha tênue entre a segurança e o caos total.

O ano é 2010, e você é o Fiscal dos Fiscais. O seu trabalho não é só difícil — é vital! Sua missão é documentar cada verificação feita por seus fiscais para que, se houver qualquer deslize, seu superior possa tomar as devidas providências rapidamente.

Com um olhar atento, você guarda as informações em um dicionário, pois são relatórios confidenciais de inteligência. As chaves desse dicionário são os nomes dos fiscais, e os valores são listas de verificações, cada uma contendo o nome do suposto policial fiscalizado e o resultado da fiscalização (se foi realizada corretamente ou não).

A tensão aumenta durante o turno, pois qualquer erro pode significar que um inimigo se esgueirou pela base — e cabe a você garantir que todos estão fazendo seu trabalho de maneira impecável.

Input:

Ao iniciar o seu trabalho, você deve registrar o nome do primeiro fiscal, podendo conter letras, números e/ou caracteres especiais (afinal, é um nick de um jogo):

N1CK$F1SC4L

Em seguida, informe quantos policiais foram fiscalizados pelo fiscal:

X (número inteiro)

Depois, colete o nome do primeiro policial fiscalizado:

N1CK$F1SC4L1Z@D@

Adiante, informe se a fiscalização foi feita corretamente ou incorretamente:

CORRETA ou INCORRETA

Faça os dois últimos passos pela quantidade de policiais fiscalizados, inserindo sempre informações de novas tentativas de entrada na base. Considere que nenhum policial tentará entrar na base mais de uma vez.

Ao fim, caso haja outro fiscal durante o seu turno, repita o processo. Caso não tenha mais nenhum fiscal, encerre o seu turno enviando:

FIM DO TURNO

Output:

Quando o sistema for iniciado, uma mensagem de boas-vindas deve ser impressa:

Ó, Fiscal dos Fiscais, seja bem-vindo(a) ao seu turno!

Sempre que houver um novo fiscal a ser fiscalizado:

Qual o fiscal que será fiscalizado agora?

Sempre que for necessário informar a quantidade de policiais fiscalizados pelo fiscal:

Quantos (supostos) policiais o fiscal fiscalizou?

Sempre que for necessário coletar o nome de um policial fiscalizado pelo fiscal:

Qual o nome do {x}º policial fiscalizado pelo fiscal?

Sempre que for necessário informar se a fiscalização foi feita corretamente ou incorretamente:

Qual o resultado desta fiscalização?

Sempre que o turno for encerrado (pule uma linha para imprimir):

O meu turno acabou. Agora, vou apresentar os relatórios ao meu chefe.

Se não houve fiscal para ser fiscalizado (pule uma linha para imprimir):

Hoje o dia foi tão tranquilo como um esquilo, sem fiscais para serem fiscalizados.

Quando não houver mais fiscais para serem fiscalizados e houver, ao menos, um fiscal (pule uma linha para imprimir):

fiscal_1 fiscalizou x supostos policiais, que foram:
fiscalizado_1, que foi fiscalizado corretamente.
fiscalizado_2, que foi fiscalizado incorretamente.
fiscalizado_n, que foi fiscalizado corretamente.

fiscal_2 teve um turno de boa na lagoa, sem supostos policiais para serem fiscalizados.

fiscal_n fiscalizou y supostos policiais, que foram:
[...]

Quando todas as informações forem impressas corretamente (pule uma linha para imprimir):

Ufa! Mais um dia finalizado sem o meu chefe me dar um pé no Habbo.

Solução Possível (em Python):

turno = True
infos_fiscais = {}

print('Ó, Fiscal dos Fiscais, seja bem-vindo(a) ao seu turno!')

while turno:
    print('Qual o fiscal que será fiscalizado agora?')
    fiscal = input()

    if fiscal not in infos_fiscais.keys() and fiscal != 'FIM DO TURNO':
        infos_fiscais[fiscal] = []

    if fiscal != 'FIM DO TURNO':
        print('Quantos (supostos) policiais o fiscal fiscalizou?')
        qtd_policiais = int(input())

        for ordem in range(0, qtd_policiais):
            print(f'Qual o nome do {len(infos_fiscais[fiscal]) + 1}º policial fiscalizado pelo fiscal?')
            nome_policial = input()

            print('Qual o resultado desta fiscalização?')
            resultado = input()

            infos_fiscais[fiscal].append([nome_policial, resultado])
    else:
        turno = False

print('\nO meu turno acabou. Agora, vou apresentar os relatórios ao meu chefe.')

if len(infos_fiscais) == 0:
    print('\nHoje o dia foi tão tranquilo como um esquilo, sem fiscais para serem fiscalizados.')
else:
    for fiscal in infos_fiscais.keys():
        if len(infos_fiscais[fiscal]) == 0:
            print(f'\n{fiscal} teve um turno de boa na lagoa, sem supostos policiais para serem fiscalizados.')
        else:
            print(f'\n{fiscal} fiscalizou {len(infos_fiscais[fiscal])} supostos policiais, que foram:')

            for ordem_fiscalizado in range (0, len(infos_fiscais[fiscal])):
                print(f'{infos_fiscais[fiscal][ordem_fiscalizado][0]}, que foi fiscalizado', end=' ')
                if infos_fiscais[fiscal][ordem_fiscalizado][1] == 'CORRETA':
                    print('corretamente.')
                else:
                    print('incorretamente.')

print('\nUfa! Mais um dia finalizado sem o meu chefe me dar um pé no Habbo.')
